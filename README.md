# Helpful commands

Run Tinytalk on a file:

```
deno run --allow-read tinytalk.ts hello.tiny
```

Run Tinytalk REPL:

```
deno run tinytalk.ts
```

Generate docs:

```
deno doc read.ts
```

Run tests:

```
deno test
```

Format everything:

```
deno fmt
```
